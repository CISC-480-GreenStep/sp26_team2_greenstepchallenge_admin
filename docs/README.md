# Documentation

Project documentation for the GreenStep Sustainability Challenge Admin Console.

## Start here

- **[`ARCHITECTURE.md`](./ARCHITECTURE.md)** — annotated source tree, "where do I
  change X?" recipes, and "how do I add Y?" playbooks. Read this first when
  onboarding or before opening a refactor PR.

  file layout and navigation).

  (pre-merge snapshots, etc.).
