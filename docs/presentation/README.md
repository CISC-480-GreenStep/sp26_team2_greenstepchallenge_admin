## Workflow

1. **Read `slides-outline.md`** — has [TBD] placeholders for presenter
   assignments and time-per-section.
2. **Split presenter assignments as a team**, replace every [TBD: presenter]
   with a name. Aim for the 15-min budget the rubric requires.
3. **Personalize each slide's bullets** — tone, emphasis, what to skip.
4. **Build the actual deck** in Slides/Keynote/PowerPoint. Export to
   `Team2_ProjectPresentation.pdf` (per the rubric's file-naming rule)
   and submit to Canvas.
5. **Rehearse with `demo-script.md` and `qa-prep.md` open** so demo timing
   and Q&A reflexes are fresh.
